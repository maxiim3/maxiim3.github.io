# Project Audit — 2026-02-21

## 1. Project Brief

**What:** Site CV/Portfolio personnel pour GitHub Pages — Maxime Tamburrini, Développeur Front-End Vue/Nuxt.

**Goal:** Transformer un parcours atypique (15 ans musique/son, restauration, 4 langues) en avantage compétitif pour le marché tech. Objectifs mesurables :
- 3+ entretiens techniques en 2 mois
- 1+ demande freelance via le site en 3 mois
- Améliorer le taux de réponse aux candidatures de 50%

**Stack:** Astro 5.17.1, TypeScript strict, Bun, vanilla CSS (CSS Layers), GitHub Pages, GitHub Actions CI/CD.

**Concept clé:** Toggle CSS — le site se présente d'abord en HTML brut (raw), puis une transition radiale (View Transitions API) révèle le design complet. C'est le "wow moment" qui démontre la maîtrise du progressive enhancement.

## 5. Critical Issues Found

### BLOCKER
1. **NFR3 FAIL — Page weight exceeds 200KB budget by ~120KB.** The Inter variable font (291KB) alone exceeds the budget. Options: subset the font (only used weights/chars), use a lighter font, or revise the budget.

### HIGH
2. **FR5 PARTIAL — Projects section contains PLACEHOLDER DATA.** The fr.ts/en.ts files have sample projects ("Portfolio Vue 3", "Dashboard Analytics", "Component Library", "E-commerce Platform") marked with comment "Sample project data for demonstration. Update with real projects before deployment." This is visible on the live site.

3. **base.css exceeds <1KB architecture spec.** Architecture document specified base.css should be <1KB render-blocking. Actual: 7.7KB (source), ~3KB (minified). This is 3x the spec. Contains styles beyond the minimum needed for raw state.

### MEDIUM
4. **FR33 — Site not deployed to production.** Working on `v2` branch. deploy.yml triggers on push to `main`. Need to merge v2 → main to trigger first deployment.

5. **Case study results may be inferred.** Agent-generated metrics in case study data may need verification against real figures.

6. **Sprint tracker drift.** Stories 5-1, 5-2, 6-1 are functionally complete but tracked as "in-progress". No backlog stories remain — the project has no planned next work.

### LOW
7. **Case study screenshots array is empty.** Data structure supports screenshots but none are provided.
8. **No Lighthouse audit has been run** against the built site.
9. **CSS files live in public/styles/ instead of src/styles/** as architecture specified.
